# The Garden Group Service Desk System
This application is designed for employees of an organization to call in problems with their devices, which are registered as service tickets. The goal of this application is to streamline the process of reporting and resolving device issues within the organization.

## Contributors
This project was completed by five 2nd year students in September and October 2022. I contributed the code for User Management, Add User, Dashboard, and Archiving Tickets (additional functionality) parts of the system.

## Grade
The students received a grade of 88/100 for this project.

## Technology
This project was completed to practice NoSQL and C# knowledge. The database used in this project is MongoDB.

## Features
The Service Desk Ticketing System includes features for user management, adding new users, a dashboard for tracking and managing tickets, and archiving tickets for record-keeping purposes.
